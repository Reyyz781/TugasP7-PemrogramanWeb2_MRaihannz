# Tugas P7 - Pemrograman Web 2
## Aplikasi MVC Manajemen Data Mahasiswa

### Nama: Muhammad Raihan Nur ZamZam
### NPM: 2411700030

### Fitur:
- ✅ Create, Read, Update, Delete data mahasiswa
- ✅ MVC Architecture
- ✅ Database MySQL
- ✅ Bootstrap UI

### Cara Menjalankan:
1. Jalankan XAMPP (Apache & MySQL)
2. Import database `db_mahasiswa.sql`
3. Akses: `http://localhost/mvc-app/public/?action=index`

### Struktur URL:
- Daftar: `/?action=index`
- Tambah: `/?action=tambah` 
- Edit: `/?action=edit&id=1`
- Detail: `/?action=detail&id=2`
